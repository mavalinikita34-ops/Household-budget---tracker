# Household-budget---tracker
### Tools: Microsoft Excel

## 📌 About Project
A monthly household budget tracker built in Excel 
to monitor income, expenses and savings throughout 
the year 2025.

## 📊 What's Inside
- Monthly Data Sheet — Jan to Dec complete data
- Pivot Analysis — Category-wise & monthly summary
- Dashboard — Charts & KPI cards

## ✨ Key Features
- Total Income, Expense & Savings tracking
- Category-wise expense breakdown
  (Rent, Groceries, Electricity, Transport, 
  Medical, Entertainment, Clothing, Misc)
- Monthly Savings % calculation
- Best Savings Month identified — March

## 📐 Formulas Used
- SUM — Total expense calculation
- IF — Savings percentage
- Cross-sheet references

## 📈 Key Insights
- Total Annual Savings — Rs. 1,21,700
- Best Month — March (Rs. 12,400 savings)
- Highest Expense Month — December
- Average Savings Rate — 35%

## 🛠️ Skills Demonstrated
- Data Entry & Organization
- Excel Formulas & Functions
- Dashboard Creation
- Data Analysis & Visualization
